# Investor Showcase: Technology Stack

The modern core technologies powering the FedCore and FedSOC platforms.

---

## 1. Federated Learning & Machine Learning Core

- **Flower Framework**: Coordinates the secure gRPC server-client nodes aggregation cycles.
- **Scikit-Learn (SGDClassifier)**: Core local machine learning model with incremental training capability.
- **XGBoost (XGBClassifier)**: Alternate tree-based adapter for high-precision local fit runs.
- **Python**: Core scripting language used to drive preprocessors and database mappings.

---

## 2. Telemetry and Presentation Dashboard

- **Next.js 16 (Turbopack)**: Web application server serving fast static and dynamic pages.
- **React 19 & TypeScript**: Provides modular component architectures with strict typing constraints.
- **TailwindCSS v4**: Next-generation utility-first CSS framework hosting design system variables.
- **Recharts**: Responsive charting library depicting convergence curves, drift vectors, and bandwidth consumption.

---

## 3. Database & Orchestration

- **PostgreSQL (Supabase)**: Real-time database tracking nodes online timers and round summaries.
- **Ollama API**: Interface for running Llama 3.1/3.3 models locally inside internal SOC infrastructures.
